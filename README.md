## Requirements

```
python 3.11.0
django 4.0.0
Pillow 1.0.0
```

## Hướng dẫn

Để tạo tài khoản admin:
```
python3 manage.py createsuperuser
```

Để chạy project:
```
python3 manage.py runserver <port>
```
